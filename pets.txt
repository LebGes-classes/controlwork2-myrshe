1 ооп это, когда в программе создают объекты которые взаимодейтсвуют между собой
есть школьный класс, это один класс, есть класс учеников, учителей и тд, они все взаимодействуют между собой
2 основные принципы ооп, три кита(инкапсуляция, наследование, полиморфизм)
наследование: когда от одного класса(родительского) другому классу (дочернему) наследуются переменные
полиморфизм: семейство кошачих: кошки, рыси, тигры, львы. они все разные но у каждого есть уши, нос, усы и тд это 
3 инкапсуляция создана для того чтобы нельзя было изменять какие-то переменные в классе
допустим мы создали класс кошки, в ней есть параметр имя, инкапсуляция помогает не менять это значение
4 класс в java это методы и переменные внутри него, с помощью которых создаются объекты
пример: есть кошки- это класс, у них есть имена, цвет шерсти, вес и тд
5 отличие между классом и объектом
объект это экземпляр класса, а класс описывает переменные, методы, события и тд
пример: есть кошки- это класс, у них есть имена, цвет шерсти, вес и тд
а есть конкретная кошка по имени Нюша(она объект или экзмепляр класса кошки)
